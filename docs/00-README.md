# 00 — 项目总览

## 产品定位

VocabMaster 是一款基于艾宾浩斯遗忘曲线的分级背单词应用，目标用户是备考 KET/PET/FCE/CAE 等剑桥英语考试，以及国内小学—专八各学段的英语学习者。

## 核心价值主张

1. **科学复习**：严格按艾宾浩斯九阶段间隔重复（SRS）安排复习，不是伪 SRS。
2. **分级清晰**：十个等级各自独立词库，用户可同时学多个等级。
3. **多端同步**：Web + 小程序 + Android，进度实时同步。
4. **离线可用**：下载词库后无网也能学。
5. **数据可视化**：学习报表 + 个人遗忘曲线叠加展示。

## 十个等级

| 等级 | 预期词汇量 | 目标人群 |
|------|-----------|---------|
| KET (剑桥通用 A2) | ~1500 | 初级学习者、小学高年级 |
| PET (剑桥通用 B1) | ~3500 | 中考水平 |
| FCE (剑桥通用 B2) | ~5000 | 高考 / 大学入门 |
| CAE (剑桥通用 C1) | ~7500 | 雅思 7+ / 托福 100+ |
| 小学 | ~800 | 小学 1-6 年级课标 |
| 初中 | ~2000 | 初中课标 |
| 高中 | ~3500 | 高中课标 |
| 大学四级 (CET-4) | ~4500 | 大学四级 |
| 大学六级 (CET-6) | ~6000 | 大学六级 |
| 专业八级 (TEM-8) | ~13000 | 英语专业八级 |

总词汇量合计约 4.7 万词（含跨等级重复），实际去重后约 2.5 万独立词。

## 单词卡片字段

每张卡片至少包含：

| 字段 | 说明 | 示例 |
|------|------|------|
| word | 单词原形 | `abandon` |
| ipa | 国际音标（英式为主） | `/əˈbæn.dən/` |
| phonetic_us | 美式音标（可选） | `/əˈbæn.dən/` |
| en_definition | 英英释义 | `To leave completely and permanently.` |
| zh_definition | 中文释义（含词性） | `v. 放弃；抛弃` |
| example_en | 英文例句 | `They had to abandon the project.` |
| example_zh | 例句中文翻译 | `他们不得不放弃这个项目。` |
| topic | 主题分类 | `动作` / `情感` / `学术` |
| audio_url_uk | 英式发音音频 URL | CDN 链接 |
| audio_url_us | 美式发音音频 URL | CDN 链接 |
| image_url | 配图 URL（可选） | Unsplash 链接 |
| difficulty | 难度分（1-10） | `5` |
| frequency | 词频分（0-1） | `0.82` |

## 核心功能清单

### 学习功能
- [x] 首次学习：支持字母序 / 主题 / 随机三种顺序
- [x] 复习：按艾宾浩斯九阶段间隔自动调度
- [x] 卡片翻面：正面单词+音标+图，翻面释义+例句
- [x] 评分：认识 / 不认识（未来可扩展为模糊/一般/清晰三档）

### 测试功能
- [x] 拼写测试：看中文/听发音，拼出单词
- [x] 选择题：四选一词义
- [x] 听写模式：播放发音，输入单词
- [x] 错题本：自动收集错题专项复习

### 发音功能
- [x] 点击播放英式/美式发音
- [x] 自动播放（卡片翻面时）
- [x] 语速调节（0.5x / 1x / 1.5x）

### 激励功能
- [x] 每日打卡
- [x] 连续天数统计
- [x] 学习目标设置（每天 N 词）
- [x] 成就徽章（首次 100 词、连续 7/30 天、完成一个等级等）

### 数据功能
- [x] 今日学习报表
- [x] 周报 / 月报
- [x] 个人遗忘曲线可视化（理论曲线 + 实际复习点）
- [x] 词汇掌握热力图（按主题）

### 账号功能
- [x] 注册（手机号 / 邮箱 / 微信 / 苹果）
- [x] 登录（同上 + 第三方）
- [x] 多端实时同步
- [x] 个人配置（通知时间、每日目标、默认音标、主题）
- [x] 导出学习数据（CSV）

### 离线功能
- [x] 按等级下载完整词库到本地
- [x] 离线学习（本地进度队列）
- [x] 联网后增量同步

## 非功能需求

- **性能**：词卡接口 P95 < 200ms，复习调度计算 < 50ms
- **可用性**：99.5%（允许每月 ~3.6 小时停机）
- **数据安全**：密码 bcrypt 存储，接口 HTTPS，敏感字段（手机号）入库时 AES 加密
- **合规**：遵循《个人信息保护法》，提供数据导出/删除接口
- **国际化**：界面文案 i18n，初期中英双语
- **兼容性**：Web 端支持 Chrome/Edge/Safari 最近两个大版本，Android ≥ 8.0
